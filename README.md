# movie-filter-app

## Description

L'application Movie-Filter-App est un outil qui permet de choisir rapidement le film à visionner.

Le projet est pensé pour être un moyen rapide et plus maîtrisé de sélectionner son film.

Sur les plateformes de streaming, l'utilisateur est souvent perdu face à une proposition pléthorique et généralement orientée par les algorithmes.

Ce projet a été l'occasion de mettre en pratique Redux, de découvrir les subtilités entre les versions anciennes et la version la plus moderne ; également une occasion d'explorer les outils de développement tels que React/Redux DevTools.

## Cararactéristiques

L'application Movie-Filter-App a trois caractéristiques essentielles :

- Le filtrage par catégorie,
- Le filtrage par film,
- L'accès en un clic à une page détaillée, aérée et ergonomique, affichant le synopsis, le casting et la bande-annonce.

L'application offre également la possibilité de sauvegarder ses films préférés dans une liste de favoris.

## Usage

Pour lire le script :

Vérifier que Node est installé, idéalement la version 24, puis saisir les commandes suivantes :

- npm install,
- npm run dev.

## Technologies utilisées

- Vite
- Redux
- React
- Rect-Router

## En Perspective

Le projet est toujours en cours de développement. Sa version finale inclura des fonctionnalités complémentaires/ supplémentaires, en plus d'un design basé sur le prototype initial, ergonomique et répondant aux exigences d'accessibilité.

En matière de fonctionnalités, par exemple, le champ de recherche permettra aussi d'effectuer une recherche par le nom des acteurs du film.
